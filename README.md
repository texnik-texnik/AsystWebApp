# Ассистент (PWA Quiz App)

**Ассистент** — это современное мобильное PWA-приложение (Progressive Web App) для подготовки к экзаменам на базе текстовых файлов. Приложение полностью автономно, работает без бэкенда и позволяет превратить любой текстовый список вопросов в интерактивный тренажер.

## 🚀 Основные возможности

- **Полный Offline**: Благодаря Service Worker приложение работает без интернета после первого открытия.
- **Хранилище IndexedDB**: Все ваши тесты и результаты сохраняются в памяти браузера. Можно загрузить десятки файлов.
- **Три режима работы**:
  - **Экзамен**: Случайный порядок вопросов, выбор количества вопросов, мгновенная проверка ответов, работа над ошибками и финальный результат.
  - **Просмотр (Обучение)**: Порядок как в файле, правильные ответы всегда подсвечены, удобная навигация и прыжки по номерам вопросов. Сохранение прогресса для каждого теста.
  - 🔥 **Работа над ошибками**: Специальный режим, позволяющий повторять только те вопросы, в которых вы ошиблись ранее.
- **Умный Парсер**: Устойчив к ошибкам форматирования, поддерживает многострочные вопросы, корректно обрабатывает кириллицу (UTF-8) и не удаляет вопросы без отмеченного правильного ответа.
- **Темная тема**: Автоматическое и ручное переключение темы для комфортной учебы в любое время суток.
- **Резервное копирование**: Функции Экспорта и Импорта всей базы тестов вместе со статистикой в JSON-файл.
- **Тактильный отклик**: Вибрация при правильных и неправильных ответах для лучшего запоминания.
- **Поиск и Статистика**: Мгновенный поиск по каталогу, отслеживание рекордов и прогресса изучения каждого теста.

## 📝 Формат текстового файла

Приложение понимает файлы `.txt` со следующей структурой:

```text
? Текст вопроса (может занимать несколько строк)
+ Правильный вариант ответа
- Неправильный вариант ответа
- Еще один неправильный вариант

? Следующий вопрос...
+ Правильный ответ
- Неправильный ответ
```

## 🛠 Установка и запуск

### Как использовать (PWA):
1. Откройте приложение в браузере вашего смартфона.
2. В меню браузера выберите **"Добавить на главный экран"** (Install App).
3. Теперь приложение будет запускаться как обычная программа с красивой иконкой и работать без интернета.

### Локальный запуск (для разработчиков):
Если вы используете Termux или ПК с Python:
```bash
python3 -m http.server 8081
```
Затем откройте `http://localhost:8081`.

### Проверка кода:
В проект добавлены простые проверки без внешних зависимостей:
```bash
npm run validate
```
Команда проверяет синтаксис JavaScript-файлов и запускает тесты парсера.
